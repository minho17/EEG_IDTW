Supplementary Materials
Paper Title: "EEG-based Biomarker for the Identification of Mild Cognitive Impairment via Coherency Analysis and Iterative Dynamic Time Warping"

* Code and Data Descriptions
  - apply_idtw.m: MATLAB implementation of the IDTW transformation, the core component of the proposed method (refer to Algorithm 2 in the manuscript).
  - biomarker.mat: Data file containing the extracted biomarkers proposed in this study.
  - example.py: Python example script demonstrating how to load the biomarkers and perform statistical analysis (t-test).

* Software Environment and Requirements
  - MATLAB: R2023a
  - Python: 3.13.5
  - SciPy: 1.16.1
